# Architecture

## System Design
This is a React application built with modern web technologies, focusing on AI-driven visualization.

## Technology Stack
| Layer | Technology |
|-------|------------|
| Frontend | React 18 |
| Build Tool | Vite |
| UI Components | Shadcn UI |
| Styling | Tailwind CSS |
| Language | TypeScript |
| Visualization | React Flow |
| Animation | Framer Motion |

## Key Components
- **Layout**: Global wrapper with responsive navigation and footer.
- **TextInputSection**: User interface for entering text and triggering AI analysis.
- **MindMapViewer**: Interactive canvas for rendering hierarchical data using React Flow.

## Design Patterns
1. **Component Structure**: Feature-based organization.
2. **State Management**: React hooks for local UI state and TanStack Query for API interactions.
3. **Styling**: Utility-first with Tailwind CSS and Framer Motion for smooth transitions.
